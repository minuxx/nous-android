# Nous
업무 일지 기록 서비스

<br/>

## 주요 기능
- 업무 일지 조회 / 작성 / 삭제
- 채용 뉴스 조회

<br/>

## 개발 환경
- Kotlin based, Coroutines + Flow for asynchronous.
- Minimum SDK level 24

### Jetpack
- Lifecycle
- ViewModel
- DataBinding
- Navigation
- Room
- Hilt

### Library
- Material Desgin3
- ksp
- Retrofit2 & OkHttp3
- Naver Search Open API
- Glide
- Gson

### Architecture
- UI Layer -> Domain Layer -> Data Layer
- MVVM Architecture (View - ViewModel - Model)
- Repository Pattern
