## 주요 기능
- 업무 일지 조회 / 작성 / 삭제
- 채용 뉴스 조회

<br/>

## 개발 환경
- Kotlin based, Coroutines + Flow for asynchronous.
- Minimum SDK level 24

### Architecture
- UI Layer -> Domain Layer -> Data Layer
- MVVM Architecture (View - ViewModel - Model)
- Repository Pattern
- Single Activity Architecture

### Jetpack
- Lifecycle
- ViewModel
- DataBinding
- Navigation
- Room
- Hilt

### Library
- Material Desgin3
- ksp
- Retrofit2 & OkHttp3
- Glide
- Gson

### Open API
- Naver Search Open API
